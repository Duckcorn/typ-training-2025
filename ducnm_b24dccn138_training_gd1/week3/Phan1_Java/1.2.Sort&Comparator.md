# 1.Sort
> Phương thức Arrays.sort() trong Java được sử dụng để sắp xếp các phần tử của một mảng.
> - Nó cung cấp nhiều sự lựa chọn sắp xếp toàn bộ mảng, mảng con, hoặc sắp xếp các đối tượng tùy chỉnh sử dụng comparators
> - Nó có thể sắp xếp cả mảng dữ liệu "primitive"(int, char, etc,..) và mảng dối tượng (String,etc,...)

**Ví dụ**
```java
import java.util.Scanner;
import java.util.Arrays;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int[] arr1 = {2, -1, 3, 4};
        char[] arr2 = {'b', 'a', 'c', 'b'};
        Arrays.sort(arr1);
        Arrays.sort(arr2);
        System.out.println(Arrays.toString(arr1)); // -> -1 2 3 4
        System.out.println(Arrays.toString(arr2)); // -> a b b c
        scanner.close();
    }
}
```
**Giải thích**
- Arrays.sort() sắp xếp các phần tử theo thứ tự tăng dần
- Các phần tử giống nhau không được loại bỏ
- "Primitive arrays" thì không thể dùng sắp xếp tùy chỉnh(custom comparators)


## Syntax của Arrays.sort()
### 1.Để sort toàn bộ mảng
```java
Arrays.sort();
```

### 2. Để sort 1 mảng con
```java
public static void sort(int[] arr, int from_Index, int to_Index) ;
```

**Ví dụ**
```java
import java.util.Scanner;
import java.util.Arrays;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int[] arr = {2, -1, 4, 3};
        Arrays.sort(arr, 1, 4);
        System.out.println(Arrays.toString(arr)); // -> 2 -1 3 4
        scanner.close();
    }
}
```
**Giải thích** : chỉ có phần tử ở chỉ số 1,2,3 được sắp xếp, phần tử đầu tiên thì giữ nguyên

### 3. Sắp xếp giảm dần
- Để sắp xếp một mảng theo thứ tự giảm dần, ta có thể sử dụng phương thức Arrays.sort() kết hợp với Collections.reverseOrder() làm bộ so sánh (comparator).
```java
import java.util.Scanner;
import java.util.Arrays;
import java.util.Collections;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Integer[] arr = {2, -1, 3, 4};
        Arrays.sort(arr, Collections.reverseOrder());
        System.out.println(Arrays.toString(arr)); // -> 4 3 2 -1
        scanner.close();
    }
}
```

### 4. Custom Sorting với Comparator
```java
import java.util.*;

// Custom class
class Student{
    int roll;
    String name;
    String address;

    Student(int roll, String name, String address){

        this.roll = roll;
        this.name = name;
        this.address = address;
    }

    // Print student details
    public String toString() {
        return roll + " " + name + " " + address;
    }
}
// Comparator to sort by roll number
class SortByRoll implements Comparator<Student>{
    public int compare(Student s1, Student s2){

        return s1.roll - s2.roll;
    }
}
class Geeks {
    public static void main(String[] args){

        Student[] students = {
                new Student(1, "Ram", "MP"),
                new Student(2, "Shyam", "UP"),
                new Student(3, "Hari", "Delhi")
        };
        // Sort using custom comparator
        Arrays.sort(students, new SortByRoll());
        // Print sorted students
        for (Student s : students)
            System.out.println(s);  // -> 1 Ram MP
                                          2 Shyam UP
                                          3 Hari Delhi
    }
}
```

**Giải thích**
- Comparator cho phép định nghĩa logic sắp xếp tùy chỉnh mà không cần chỉnh sửa lớp (class) gốc.
- Trong ví dụ này, các sinh viên được sắp xếp theo số báo danh (roll number).


# 2. Comparator
> Giao diện Comparator trong Java được sử dụng để định nghĩa logic sắp xếp tùy chỉnh cho các đối tượng. Nó thuộc gói java.util và cho phép sắp xếp các đối tượng của lớp do người dùng định nghĩa mà không cần chỉnh sửa mã nguồn của lớp đó. Giao diện này đặc biệt hữu ích khi:
> - Cần sắp xếp cùng một lớp theo nhiều tiêu chí khác nhau (ví dụ: theo tên, tuổi, điểm số, v.v.).
> - Chúng ta muốn giữ cái logic sắp xếp riêng biệt so với khai báo class.

## Syntax
```java
class MyComparator implements Comparator<Type> {
           public int compare(Type obj1, Type obj2) {
               // comparison logic
      }
}
```
- Nó sẽ trả về một số âm nếu obj1 < obj2
- Nó sẽ trả về số 0 nếu obj1 = obj2
- Nó sẽ trả về một số dương nếu obj1 > obj2

**Ví dụ**
```java
class SortByName implements Comparator<Student> {
    public int compare(Student s1, Student s2) {
        return s1.name.compareTo(s2.name);    // -> Dùng để sort theo tên
    }
}
```
